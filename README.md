# Beats Market

## Регистрация
1. Нажмите REGISTER
2. Введите имя, email и пароль

## Покупка битов
1. Зайдите в SHOP
2. Нажмите на обложку для прослушивания
3. Нажмите ADD TO CART
4. Перейдите в корзину и оплатите

## Лицензии
1. Нажмите BUY LICENSE
2. Выберите тип лицензии: WAY, TRACKOUT или EXCLUSIVE

## Админ панель (для администраторов)
- Доступна в профиле
- Загрузка битов
- Управление пользователями
- Статистика
